¿Qué son las Media Queries?

Las media queries permiten aplicar estilos CSS dependiendo del dispositivo, como el ancho de pantalla, orientación o resolución.

Son esenciales para el diseño responsivo.

  • Adaptación de contenido
  • Mejor experiencia de usuario
  • Optimización en móviles

Sintaxis Básica

@media screen and (max-width: 768px) {
    .elemento {
        background-color: blue;
    }
}

Ejemplos para PC, Tablet y Móvil

PC Desktop

@media (min-width: 1024px) {
  .container {
    width: 1200px;
  }
}

Tablet

@media (max-width: 1023px) {
  .container {
    width: 90%;
  }
}

Móvil

@media (max-width: 767px) {
  .container {
    padding: 10px;
  }
}
