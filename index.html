Diseño Responsivo para la Web Moderna

¿Qué son las Media Queries?

Las Media Queries son una característica fundamental de CSS3 que permite aplicar estilos condicionalmente basados en las características del dispositivo.

Tip: Las Media Queries son la base del diseño web responsivo, permitiendo que tu sitio se adapte a diferentes dispositivos automáticamente.

Propósito Principal

  • Adaptar el diseño a diferentes tamaños de pantalla
  • Mejorar la experiencia del usuario
  • Aumentar la accesibilidad
  • Optimizar el rendimiento en dispositivos móviles

Sintaxis Básica

@media screen and (max-width: 768px) {
  /* Estilos para pantallas de 768px o menos */
  .elemento {
    background-color: blue;
  }
}

Componentes Clave

Media Type

screen, print, speech

Media Features

width, height, orientation

Logical Operators

and, not, only, comma

Ejemplos Prácticos

PC Desktop

@media screen and (min-width: 1024px) {
  .container {
    width: 1200px;
    grid-template-columns: 1fr 300px;
  }
}
  • Pantallas grandes (1024px+)
  • Diseño en múltiples columnas
  • Menús horizontales

Tablet

@media screen and (max-width: 1023px) and (min-width: 768px) {
  .container {
    width: 90%;
    grid-template-columns: 1fr;
  }
}
  • Pantallas medianas (768px - 1023px)
  • Diseño adaptado para tabletas
  • Interfaz táctil optimizada

Móvil

@media screen and (max-width: 767px) {
  .container {
    width: 100%;
    padding: 10px;
  }
}
  • Pantallas pequeñas (<768px)
  • Diseño vertical
  • Botones grandes

Buenas Prácticas

Mobile First: Comienza con estilos móviles y luego añade estilos para pantallas más grandes.
Puntos de quiebre comunes:
  • Móvil hasta 768px
  • Tablet 768px - 1024px
  • Desktop 1024px+
Consejos adicionales:
  • Evita el zoom automático en móviles
  • Prueba en múltiples dispositivos y navegadores
  • Optimiza imágenes para diferentes resoluciones
